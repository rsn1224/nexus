# Storage Wing（ストレージ管理） — 仕様書

## 概要
ドライブ一覧の健全性表示とディスククリーンアップツールを提供するストレージ管理画面。各ドライブの使用状況を可視化し、不要ファイルの削除を支援する。

## 前提条件
- バックエンドがドライブ一覧（名前、モデル、ファイルシステム、使用量、ヘルス）を取得できること
- 一時ファイル、ごみ箱、システムキャッシュの削除権限があること
- ディスク分析（ファイルパス一覧の取得）が実行可能であること

## シナリオ

### DriveList

#### Scenario 1: ドライブ一覧の表示
- **Given** Storage Wing が表示される
- **When** ドライブデータが取得済み
- **Then** 各ドライブがカード形式で表示され、名前、ヘルスステータス、モデル、ファイルシステム、使用量（使用中/合計 GB）、プログレスバーが含まれる

#### Scenario 2: ヘルスステータスの色分け — Good
- **Given** ドライブの使用率が 75%
- **When** ヘルスステータスが評価される
- **Then** 緑色（Good）で表示される（< 80%）

#### Scenario 3: ヘルスステータスの色分け — Warning
- **Given** ドライブの使用率が 85%
- **When** ヘルスステータスが評価される
- **Then** 黄色（Warning）で表示される（> 80%）

#### Scenario 4: ヘルスステータスの色分け — Critical
- **Given** ドライブの使用率が 95%
- **When** ヘルスステータスが評価される
- **Then** 赤色（Critical）で表示される（> 90%）

#### Scenario 5: ドライブの分析実行
- **Given** ドライブカードが表示中
- **When** 分析ボタンを押す
- **Then** ディスク分析が実行され、ANALYSIS セクションにファイルパス一覧が表示される

### CleanupPanel

#### Scenario 6: 一時ファイルのクリーンアップ
- **Given** CleanupPanel が表示中
- **When** TEMP FILES ボタンを押す
- **Then** 一時ファイルが削除され、解放されたバイト数が緑色で SCAN RESULTS に表示される

#### Scenario 7: ごみ箱のクリーンアップ
- **Given** CleanupPanel が表示中
- **When** RECYCLE BIN ボタンを押す
- **Then** ごみ箱が空にされ、解放されたバイト数が緑色で表示される

#### Scenario 8: システムキャッシュのクリーンアップ
- **Given** CleanupPanel が表示中
- **When** SYSTEM CACHE ボタンを押す
- **Then** システムキャッシュが削除され、解放されたバイト数が緑色で表示される

#### Scenario 9: フルクリーンアップ
- **Given** CleanupPanel が表示中
- **When** FULL CLEANUP ボタンを押す
- **Then** 一時ファイル、ごみ箱、システムキャッシュがすべて削除され、合計解放バイト数が表示される

### StorageInfo

#### Scenario 10: ストレージ概要の表示
- **Given** ストレージデータが取得済み
- **When** StorageInfo がレンダリングされる
- **Then** 合計容量、使用率（%）、空き容量が表示される

## エッジケース
- ドライブが 0 台検出された場合、エラーメッセージを表示する（通常ありえないが防御的に処理）
- クリーンアップ対象が 0 バイトの場合、「クリーンアップ不要です」と表示する
- クリーンアップ中に権限エラーが発生した場合、スキップされたファイル数を報告する
- ディスク分析が大量のファイルを返す場合、ページネーションまたは仮想スクロールで表示する
- ネットワークドライブは一覧に含めない
- リムーバブルドライブ（USB 等）は接続時に動的に追加される

## 非機能要件
- ヘルス閾値: Critical（使用率 > 90%）、Warning（> 80%）、Good（≤ 80%）
- クリーンアップ処理中はプログレスインジケーターを表示する
- ディスク分析は非同期で実行し、UI スレッドをブロックしない
- 解放バイト数は適切な単位（KB/MB/GB）で自動フォーマットする
- クリーンアップ前に確認ダイアログを表示する（FULL CLEANUP は特に警告を強調）
